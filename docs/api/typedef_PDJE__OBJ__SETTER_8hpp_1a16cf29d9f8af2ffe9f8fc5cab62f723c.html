Typedef OBJ_SETTER_CALLBACK

Typedef Documentation

using OBJ_SETTER_CALLBACK = std::function<void(const std::string, const uint16_t, const std::string, const std::string, const std::string, const unsigned long long, const unsigned long long, const uint64_t)>

note object setter callback



The function passes only the arguments for creating an object. Created objects must be stored in global variables or variables captured by

reference when creating a lambda function.

Args describe

1 noteType, the note type. e.g) hold, tap, slide

2 noteDetail, the detail

3 firstArg

4 secondArg

5 thirdArg

1~5 these are meta datas for note.

6 Y axis position, the first logical position of note 7 Y axis position 2, the second logical position of note, use for long notes or else
